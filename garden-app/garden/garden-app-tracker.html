Garden
SwiftUI · SwiftData · CloudKit · iOS 17+
1
CloudKit Dashboard — Deploy Schema to Production
icloud.developer.apple.com → container iCloud.com.drivercyber.garden → "Deploy Schema Changes." Required before any TestFlight or App Store build, because production builds connect to Production CloudKit (dev builds connect to Development, which auto-creates schema and is why ⌘R "just works" on Chad's phone today). Button location is unclear in the redesigned Console UI — try Settings in the left sidebar, the per-environment view, or a per-record-type action menu.
2
Host the privacy policy
Public App Store listings require a hosted privacy policy URL. Draft is already written in app-store-listing.md. Easiest paths: GitHub Pages on a Driver-cyber repo, or a published iCloud note. This was deferred when the plan was TestFlight Internal; with the App-Store-first pivot it's load-bearing.
3
Capture 5 screenshots (6.7" iPhone display)
App Store requires the 6.7" set; 6.1" is recommended too. Set of 5 sketched in app-store-listing.md: composer, populated notes list, category chips, Calm screen with checkbox, Calm screen with confetti. Capture from iPhone 16e (Volume Up + Side button) or simulator (Cmd-S in iOS Simulator).
4
App Store Connect record + Privacy nutrition label
appstoreconnect.apple.com → My Apps → "+" with bundle ID com.drivercyber.garden (~5 min). Paste in description / keywords / subtitle / promo text from app-store-listing.md. Fill in App Privacy as "Data Not Collected" (already drafted). Add the privacy policy URL once hosted.
5
Archive + Upload to App Store Connect
Xcode → Product → Archive → Distribute App → App Store Connect → Upload (~15 min). Build appears in TestFlight + App Store sections of ASC. Submit for App Review when ready — typical wait 1–3 days for new apps; metadata rejections (privacy URL, screenshots, "doesn't do enough" risk for minimal apps) are part of the rep, not a failure mode.
6
1.0 ships → polish notes → 1.1 update → THEN share with friends
Chad's reframe (2026-04-26): submit a slightly rough 1.0 for the practice rep, fold any post-launch polish into 1.1, share the link with wife/friends only after 1.1 lands. Friends never see 1.0; they see the polished update on a real public listing.
