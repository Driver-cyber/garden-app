Test on a real device (Path A)
Plug an iPhone into the Mac, select it as the run destination, ⌘R. Validates wheat field at 60fps on real hardware, haptic feel, layout on a real screen. Also: confirm One Thing date logic across an actual midnight rollover. No App Store account required.

Apple Developer Program + CloudKit production
Enroll in Apple Developer Program ($99/yr, ~24h to activate). Then in CloudKit Dashboard, deploy the Garden schema to production — without this, real users get silent sync failures. One-click action but easy to forget.
3
App Store Connect + TestFlight
Create the App Store Connect record, archive in Xcode (Product → Archive → Distribute → Upload), wait ~5–15 min for processing, add wife as Internal Tester. She installs TestFlight, accepts the email invite, taps Install. First public-ish deployment.
